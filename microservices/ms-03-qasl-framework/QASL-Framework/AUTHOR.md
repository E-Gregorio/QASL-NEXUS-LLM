## QASL Framework (Quality Assurance Shift-Left)

Este framework representa la implementación práctica de metodologías de testing avanzadas, integrando:

| Fase | Componente | Descripción |
|------|------------|-------------|
| Phase 0 | **Unit Testing** | Vitest (128+ tests del framework de automatización) |
| Phase 1 | **Static Analysis** | Análisis de HU con IA (Claude API) |
| Phase 2 | **Recorder Pro** | Grabador de tests con 11 niveles de selectores |
| Phase 3 | **E2E Testing** | Playwright con Allure Reports |
| Phase 4 | **API Testing** | Newman/Postman Collections |
| Phase 5 | **Performance Testing** | K6 con métricas en Grafana |
| Phase 6 | **Security Testing** | OWASP ZAP Automation |
| Phase 7 | **Observability** | Loki + Promtail para logs centralizados |
