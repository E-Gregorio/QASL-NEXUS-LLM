## Sobre INGRID

**INGRID** (Intelligent Network for Generative Response Inspection & Defense) es un framework de testing de calidad para aplicaciones de Inteligencia Artificial y Modelos de Lenguaje (LLM).

### Fecha de Creación
- **Inicio del desarrollo:** Diciembre 2024
- **Primera versión:** v2.0

### Características Originales

Este framework introduce conceptos innovadores en el testing de IA:

1. **LLM-as-Judge** - Uso de Claude AI para evaluar respuestas de otros modelos (Gemini)
2. **Integración OWASP LLM Top 10** - Pruebas de seguridad específicas para LLMs
3. **Métricas en Tiempo Real** - Dashboard Grafana para visualización de resultados
4. **Automatización Playwright** - Testing automatizado de interfaces web de IA
